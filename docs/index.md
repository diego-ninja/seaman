## Overview

Seaman provides a sophisticated yet simple way to manage your Symfony development environment with Docker, offering:

- **Intelligent Project Detection**: Identifies Symfony projects using multiple indicators
- **Service Orchestration**: Manages complete Docker environments with a single configuration
- **Developer-Friendly**: Interactive CLI with beautiful output and helpful prompts
- **Type-Safe & Well-Tested**: PHP 8.4 with strict types, PHPStan level 10, 95%+ coverage

## Requirements

- Docker Desktop or Docker Engine
- Docker Compose V2
- PHP 8.3+ (for development only)
- Symfony CLI (optional, for project bootstrapping)
