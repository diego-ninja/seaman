<h3 align="center" style="border:none !important">
A Docker development environment manager for Symfony 7+
</h3>


## Overview
Seaman is sail for Symfony

## Installation

```bash
curl -sS https://raw.githubusercontent.com/diego-ninja/seaman/main/installer | bash
```

## Quick Start

```bash
# Initialize your Symfony project
seaman init

# Start your environment
seaman start
```

Your application will be available at `http://localhost`

## Install as Dev Dependency

```bash
composer require --dev seaman/seaman
vendor/bin/seaman init
```
